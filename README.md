This is a constraint solver for the game PIPS, hosted by the New York Times (https://www.nytimes.com/games/pips).

Dominoes each have two values ranging between 0 and 6.
The domino spans two tiles which may have a ruleset that excludes the placement of different values.
The goal of the puzzle is to place all dominoes in a grid of tiles so that all constraints are satisfied.

There are three difficulties provided (easy, medium, hard) which, in turn, require greater levels of logic to solve.
